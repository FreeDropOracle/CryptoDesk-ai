# Getting Started

## Local Setup

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env` and review local values.
3. Run `npm run typecheck`.
4. Run `npm test`.

## First Launch

1. Start in simulation mode.
2. Review security settings before adding API credentials.
3. Store credentials locally through the Settings page.
4. Use the Trade page for simulation-first workflows.

## Important Safety Notes

- This scaffold does not enable live exchange execution by default.
- AI outputs are recommendations, not guarantees.
- Never reuse high-privilege exchange keys for early testing.
