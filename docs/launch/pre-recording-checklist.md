# Pre-Recording Checklist

Use this checklist before recording the CryptoDesk AI beta demo.

## Workspace Prep

- [ ] Desktop cleaned of personal files
- [ ] Notifications disabled
- [ ] Browser tabs and chat tools hidden
- [ ] Time, battery, and personal account details hidden if possible

## App Prep

- [ ] Simulation mode enabled
- [ ] No real credentials present
- [ ] Market dashboard populated with demo-safe data
- [ ] AI signal card visible with reasoning
- [ ] Beta signup page ready for the CTA segment
- [ ] UI language chosen for the main recording pass

## Recording Setup

- [ ] Resolution set to 1920x1080
- [ ] Frame rate set to 30fps
- [ ] Microphone tested
- [ ] Background music chosen or intentionally omitted
- [ ] Script visible on a second screen or printed nearby

## Security And Privacy Check

- [ ] No secret keys visible
- [ ] No private email addresses visible
- [ ] No production accounts visible
- [ ] No misleading claims in overlays or spoken script

## After Recording

- [ ] Remove dead air and hesitation
- [ ] Add subtitles
- [ ] Review for accidental sensitive data exposure
- [ ] Export MP4 version for sharing
- [ ] Save final and source files in a predictable folder structure
