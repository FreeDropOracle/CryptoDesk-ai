## Message Pillars

1. Security-first desktop architecture
2. Non-custodial control over credentials
3. Explainable AI instead of black-box promises
4. Simulation-first workflows for safer onboarding

## Technical Highlights

- Electron plus React plus TypeScript
- typed preload boundary between renderer and privileged actions
- Zod validation around IPC contracts
- local encryption and OS keychain-oriented secret handling
- rate-limited exchange connectivity and advisory AI flows

## Suggested Press Description

CryptoDesk AI is a desktop crypto workflow application focused on security, transparency, and user control. It combines read-only market visibility, simulation trading, and explainable AI advisories inside a non-custodial architecture.
